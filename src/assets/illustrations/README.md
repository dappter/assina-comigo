# Diretório: Illustrations

Este diretório contém as ilustrações para Empty States (estados vazios) e de onboarding e feedback do usuário.

A preferência é manter os recortes SVGs das bibliotecas **UnDraw** colorizados a partir das cores primárias mapeadas no guia UX/UI.

Exemplos esperados:
- `empty-indications.svg`
- `success-confetti.svg`
- `error-duplicate.svg`
